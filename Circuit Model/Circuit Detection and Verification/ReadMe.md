3. The file `best(8).pt` contains the pre-trained weights for the YOLO model, which is currently the most accurate.

4. For more detailed information, please refer to `Tutorial.ipynb`.

How to read it
1. valid_circuit.ipynb drives the real-time loop, calling into the vision stack (deskewing, detection, optional video recording) and the modeling/verification layers.
2. Detection flows virtual_board_all.py → process_frame.py → pieces_location.py → add_pieces.py, which creates/updates pieces.py objects, wires them through connections.py, and logs via printscreen.py.
3. all_tests.py evaluates the current board against the skill matrices defined in tasks.py.
4. The other notebooks are utilities: test reporting, visualization experiments, gesture demos, and crop calibration.
